### BOM 浏览器对象模型

1. 弹出新的浏览器窗口的能力
2. 移动、缩放和关闭浏览器窗口的能力
3. navigator,提供想尽信息
4. location对象,提供浏览器加载页面的详尽信息
5. screen对象，提供分辨率的详尽信息
6. performance 浏览器内存占用,导航行为和时间统计的想尽信息
7. 对 cookie的支持
8. 其他自定义对象, 如XMLHttpRequest和 IE ActiveXObject
